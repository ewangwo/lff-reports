行业访谈简报

算力租赁热：真需求、真缺卡，也是真周期

一线访谈显示：推理需求拉动租赁价格，但高位入场也买下政策、折旧和残值风险。

核心判断

最赚钱的是早期低成本持卡方；新入场者不是单纯买“AI 红利”，而是在高硬件成本下赌需求、供给和残值。算力租赁的本质，是把 GPU 卡源、IDC 上架、运维、模型 API 和 Token 分发串成一门资金周转生意。

为什么涨价

  • 国内高端英伟达卡源持续紧，成规模 H100/H200/A800 集群难找。
  • 内存、硬盘涨价抬高整机成本，限制租赁价格下行空间。
  • 价格上涨既有真实缺货，也有现货溢价和周期情绪。

需求从哪里来

  • 推理成为主线：编程、短视频生成、智能体、行业应用和 Token 调用共同拉动。
  • 访谈口径：推理算力约为训练算力 3-4 倍，去年约 1.5-2 倍。
  • 大厂/公有云仍是高端 GPU 主力消耗方，小型平台更多做租赁和 Token 分发。

谁更受益

  • 早期低成本持卡方：享受租金上涨、资产重估和低折旧。
  • 英伟达生态链：CUDA、整机、组网、运维仍是商业化首选。
  • 内存、存储、IDC、电力和运维服务吃到二阶需求。

国产算力的分裂

  • 政策订单与商业 ROI 是两套逻辑；信创项目能买，不等于商业客户自愿用。
  • 访谈认为国产方案硬件成本更高、适配周期长、生态弱。
  • 关键验证不是峰值算力，而是单位调用成本、稳定性和迁移成本。

商业模式

  • 买卡或拿卡源，上架 IDC，做压测交付和运维，再接模型/API。
  • 纯租赁赚硬件周转，Token 聚合分发赚平台差价。
  • 行业关键变量是卡源、利用率、合同期限和残值。

最大风险

  • H200/H20 若放量，租赁价格短期可能下行 10-15%。
  • 高价新设备需要五年回本；一旦需求降温或残值下修，利润会迅速压缩。
  • 单一访谈偏乐观，必须用真实报价、长约比例和利用率交叉验证。
卡源 IDC 上架 模型/API Token 分发 现金回收

本质：不是单点卖铲子，而是“卡源 + 利用率 + 合同期限 + 残值”的资金周转生意。

阅读时应保持的约束

这份材料是一线行业访谈，强在细节和温度，弱在样本单一。后续需要用采购报价、IDC 合同、长约比例、真实利用率、H200/H20 到货节奏和 Token 消耗数据继续验证。它适合用于建立观察框架，不适合直接作为投资或经营决策依据。
